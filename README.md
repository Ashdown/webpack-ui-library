# webpack-ui-library
Demonstration Library for sharing UI components using webpack

## Instructions

### Run Locally

1. `npm install`
2. `npm  start`

#### Run Build

1. `npm run build`

### Use Library

1. Add to package.json `"webpack-ui-library": "git+https://github.com/Ashdown/webpack-ui-library.git#v0.0.6"`
